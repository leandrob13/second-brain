# Welcome to mdnb

A quiet, local-first Markdown notebook.

## Quick start
- Create a note with the + button.
- Create folders to organize.
- Rename or delete from the context menu.

## Markdown features supported
### Headings
# Heading 1
## Heading 2
### Heading 3

### Lists + tasks
- List item
- Another item
- [ ] Task list item
- [x] Completed task

### Bold / italic / inline code
**Bold**, *italic*, and `inline code`.

### Links
[Link](https://example.com)

### Blockquote
> This is a quote.
> It can span multiple lines.

### Horizontal rule
---

### Code blocks
```swift
print("Hello, mdnb")
```

### Tables
| Feature | Status | Notes |
| --- | --- | --- |
| Tables | Done | Markdown tables render |
| Tasks | Done | Click the checkbox |
| Images | Done | Supports URLs + files |

### Images
Images render inline from URLs and local files.

## Tips
- Notes autosave as you type.
- Sorting options live at the bottom of the sidebar.
